# DSS Rekomendasi Wisata Labuan Bajo

## Deskripsi
Project Struktur Data yang mengimplementasikan Graph sebagai Decision Support System (DSS) untuk memberikan rekomendasi rute wisata terbaik di Labuan Bajo menggunakan algoritma Dijkstra.

## Anggota Kelompok
1. Apolinarisa Jack Herment
2. Yustina Dhambo

## Teknologi
- Python
- Streamlit
- NetworkX
- Pandas

## Jenis Graph
- Weighted Graph
- Undirected Graph

## Algoritma
- Dijkstra

## Fitur
- Input lokasi wisata
- Visualisasi graph
- Perhitungan jalur terpendek
- Rekomendasi destinasi wisata

## Cara Menjalankan

pip install -r requirements.txt

streamlit run app.py
